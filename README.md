# 机器学习实验项目

> 本项目包含机器学习课程的五个核心实验，有助于加强对经典算法的理解以及应用能力。

## 实验项目
- **KNN算法** - K近邻分类器的参数优化和性能分析
- **决策树** - 决策树构建、剪枝和特征重要性分析
- **朴素贝叶斯** - 朴素贝叶斯分类器的实现与应用
- **神经网络** - CNN网络构建、训练和评估
- **模型评估与选择** - 系统的模型评估方法和选择策略

## 技术栈
- Python 3.x
- scikit-learn
- pandas, numpy
- matplotlib, seaborn

## 项目特点
✅ 完整的实验流程：数据预处理 → 模型训练 → 结果评估
✅ 详细的可视化分析：参数影响、性能对比、混淆矩阵
✅ 清晰的代码结构：模块化设计，良好注释
✅ 丰富的实验成果：准确率达到85%+，收敛稳定

## 学习收获
通过这些实验，可以掌握：
- 经典机器学习算法的原理和实现
- 模型参数调优和性能优化方法
- 数据预处理和特征工程技术
- 科研实验的规范流程和可视化方法
